# Components

This directory will contain reusable React Native components.

## Planned Components

- `CategoryCard` - Display category information
- `ItemCard` - Display rental item information
- `UserAvatar` - User profile picture component
- `RatingStars` - Rating display component
- `SearchBar` - Search input component
- `Button` - Custom button component
- `LoadingSpinner` - Loading indicator

Components will be added as the app development progresses.
